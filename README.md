# kevbee dotfiles

macOS setup: yabai + skhd + sketchybar + Übersicht + aerospace

## New machine

```bash
git clone https://github.com/kvbkvbkvb/dotfiles ~/dotfiles
cd ~/dotfiles
./install.sh
```

## First-time setup on THIS machine

```bash
cd ~/dotfiles
chmod +x setup.sh switch-theme.sh install.sh
./setup.sh
```

## Switch themes

```bash
./switch-theme.sh             # interactive picker (uses fzf if installed)
./switch-theme.sh nord
./switch-theme.sh gruvbox
./switch-theme.sh catppuccin-latte
```

## Structure

```
dotfiles/
├── install.sh          # New machine bootstrap
├── setup.sh            # First-time migration (run once on THIS machine)
├── switch-theme.sh     # Live theme switcher
├── Brewfile            # Homebrew packages (auto-generated by setup.sh)
├── themes/
│   ├── nord/
│   ├── gruvbox/
│   ├── catppuccin-latte/
│   └── Thnkdf/
└── config/
    ├── yabai/
    ├── skhd/
    ├── sketchybar/
    ├── ubersicht/
    └── aerospace/
```

## Tools

| Tool | Purpose | Config |
|------|---------|--------|
| [yabai](https://github.com/koekeishiya/yabai) | Tiling WM | `config/yabai/` |
| [skhd](https://github.com/koekeishiya/skhd) | Hotkeys | `config/skhd/` |
| [sketchybar](https://github.com/FelixKratz/SketchyBar) | Menu bar | `config/sketchybar/` |
| [Übersicht](https://tracesof.net/uebersicht/) | Desktop widgets | `config/ubersicht/` |
| [aerospace](https://github.com/nikitabobko/AeroSpace) | Alt WM | `config/aerospace/` |
